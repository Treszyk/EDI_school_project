CSV

CSV (comma-separated values) jest to typ pliku tekstowego, który zawiera dane rozdzielone seperatorami(bardzo często jest to po prostu przecinek). Pliki o tym rozszerzeniu są często używane do transferu danych z jednego programu do drugiego np. import danych między bazami danych, a arkuszami kalkulacyjnymi w excelu. Dane przechowywane są jako zwykły tekst gdzie każda pojedyńcza linia oznacza kolejny rekord tabeli.

XML

XML (Extensible Markup Language) jest to format tekstowy, który pozwana na zapisywanie danych w postaci, która jest łatwa do odczytu zarówno przez maszyny jak i przez ludzi. Dokument XML składa się z zagnieżdżonych elementów. Są one zapisywane jako znaczniki. Dzięki swojej dobrze zdefiniowanej strukturze i ogromnym zestawie narzędzi dokumenty XML są bardzo popularne co przekłada się na ich użyteczność np. przy wymianie informacji między programami.

JSON

JSON (JavaScript Object Notation) to otwarty standardowy format plików do udostępniania danych, który wykorzystuje tekst czytelny dla człowieka do przechowywania i przesyłania danych. Pliki JSON są przechowywane z rozszerzeniem .json. JSON wymaga mniej formatowania i jest dobrą alternatywą dla XML. JSON wywodzi się z języka JavaScript, ale jest formatem danych niezależnym od języka. Generowanie i parsowanie JSON jest obsługiwane przez wiele współczesnych języków programowania.
